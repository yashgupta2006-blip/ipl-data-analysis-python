# IPL Data Analysis using Python

## About Project

This is my first project uploaded on GitHub. I made this project to practice Python and Pandas by using IPL datasets.

In this project, I analyzed IPL match data and ball by ball data to find different insights related to batting, bowling, teams, toss results, and player performances.

I created this project in Jupyter Notebook as part of improving my practical skills in data analysis.

---

## Tools Used

- Python
- Pandas
- Jupyter Notebook

---

## Dataset Files

- matches.csv
- deliveries.csv

---

## Some Analysis Done

- Team with maximum wins while chasing
- Team with wins after losing toss
- Batsman with most runs
- Player who faced maximum dot balls
- Most sixes in death overs
- Bowler with most wickets
- Most wickets in death overs
- Highest powerplay partnership
- Most catches by fielders

---

## Purpose of Project

The purpose of this project was to learn how real datasets are analyzed using Python and improve my understanding of Pandas functions like filtering, grouping, sorting, and counting.
